**Дата:** 2026-05-26

## Контекст

Инструмент физически перезагружает/PoE-циклит точки доступа. Ревью с упором на blast-radius выявило кластер дыр в предохранителях: FileLock собран но не подключён, history fail-open при битом state, cap без границ и обходится из CLI. Это класс «массовое действие при отсутствующем/обходимом предохранителе».

## Находки (verified по src/)

### 🔴 Must-fix до Phase 5

1. **FileLock не подключён в рантайме.** `grep`: `FileLock` только в `services/lock.py` (определение), нигде в `src/` не используется. `cli/restart.py:52` → `svc.auto()` без lock. Cron `*/30 restart auto`: затянувшийся прогон (>30 мин) → второй инстанс видит те же кандидаты (history пишется в конце) → двойной batch.
   **Fix:** wire `FileLock` в `RestartService.auto/profile/device` (timeout=0, skip если занято). Lock path = `statefile.parent / "restart.lock"`. Новый статус `skipped_locked`.

2. **cap без границ + обход через `--max`.** `settings.py:106` `max_restarts_per_run: int = 5` — голый int. `cli/restart.py:48` пишет `--max` прямо в settings минуя валидацию → `--max -1` → срез `[:-1]` = рестарт всех кроме одного.
   **Fix:** `Field(ge=1, le=20)` на `max_restarts_per_run`; `Field(ge=0)` на cooldown'ы; валидация `--max` в CLI (reject <1).

3. **history не атомарна + fail-open.** `restart_history.py:111` docstring «Atomically save», :120 — голый `write_text`. `_load` на битом файле → `{}` → `can_restart` всем «no prior restart» → cooldown сброшен → массовый рестарт.
   **Fix:** temp-file + `os.replace` (атомарно); на битом state — **fail-closed** (`can_restart` → False «history corrupt»), не сброс; инкрементальный save после каждого record.

### 🟡 Should-fix

4. **`verify_ssl=False` по умолчанию** (`settings.py:75`). Logins поверх непроверяемого TLS → MITM с LAN. *(креды — намеренный AI-тест аккаунт, severity ниже, но secure-default правильнее.)*
   **Fix:** default `True`; UniFi self-signed → явный `verify_ssl: false` в config (example уже показывает).

5. **`legacy run` path traversal** (`legacy.py`): `script_name` сырой → `python <legacy_dir/../../any.py>`. `shell=True` нет, operator-only CLI — impact ограничен, но опасно если выставить через MCP.
   **Fix:** `Path(name).name` (strip path components) + containment check.
